# Currículo - Alan Rodrigo Pereira Freitas

Este é o código-fonte do currículo profissional de Alan Rodrigo Pereira Freitas, especializado em Cibersegurança. O currículo é desenvolvido em HTML5 e CSS3, com uso de Bootstrap para responsividade e acessibilidade. Ele também inclui boas práticas de SEO e dados estruturados JSON-LD para melhor visibilidade em mecanismos de busca.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 4
- JavaScript
- JSON-LD

## Funcionalidades

- Layout responsivo
- Links de contato direto para WhatsApp, E-mail, LinkedIn e GitHub
- Animações ao rolar a página
- Estruturação de dados para SEO (JSON-LD)

## Estrutura do Projeto

- `index.html`: Arquivo principal contendo o currículo.
- `css/style.css`: Arquivo CSS com o design e as variáveis utilizadas.
- `download/curriculo.pdf`: Arquivo PDF do currículo para download.

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em seu navegador para visualizar o currículo.
