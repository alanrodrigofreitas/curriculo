# Currículo Profissional - Alan Freitas

Este é um projeto de um currículo online desenvolvido com HTML, CSS, Bootstrap e JavaScript. O objetivo é fornecer uma apresentação clara, responsiva e acessível para o perfil profissional de Alan Freitas, tecnólogo em cibersegurança com experiência em operações industriais.

## Visão Geral

O currículo inclui as seguintes seções:
- **Objetivo**: Descrição do objetivo profissional.
- **Formação Acadêmica**: Detalhes sobre a educação e formação acadêmica.
- **Experiência Profissional**: Lista de experiências e cargos ocupados.
- **Certificações**: Certificados relevantes para a área de cibersegurança e industrial.
- **Premiações**: Reconhecimentos e prêmios profissionais.
- **Habilidades**: Competências técnicas e conhecimentos adquiridos.
- **Idiomas**: Nível de proficiência em línguas estrangeiras.
- **Download do Currículo**: Link para baixar a versão em PDF do currículo.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica para garantir acessibilidade e organização do conteúdo.
- **CSS3**: Estilização do layout, com uso de variáveis para cores e espaçamento.
- **Bootstrap 4.5**: Framework CSS para responsividade e componentes prontos, facilitando a adaptação para diferentes dispositivos.
- **JavaScript**: Uso do `IntersectionObserver` para efeitos dinâmicos de transição ao rolar a página.
- **FontAwesome**: Ícones utilizados para melhorar a visualização e acessibilidade dos links.
